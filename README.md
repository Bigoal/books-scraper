# Books.toscrape.com Scraper

A simple Python scraper that extracts book data from [books.toscrape.com](http://books.toscrape.com/).

## What it does
- Scrapes book titles, prices, ratings, availability, and links
- Handles all 50 pages of the catalog
- Saves data to CSV file
